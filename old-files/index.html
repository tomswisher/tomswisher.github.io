Background

I'm a huge fan of D3.js, the data visualization JavaScript library created by Mike Bostock, Jeffrey Heer, and a community of open source contributors.

Bostock maintains bl.ocks.org which serves as a defacto repository for the Datavis community's examples and sketches. Using his Node.js program gistup or browser extenstion you can upload a git project and host all the files you need for a modestly sized static webapp. Ian Johnson is working on a project to remove the command-line aspect and bring bl.ock-making to more users.

Helpful Resources

bl.ocksplorer.org

  • A customized search engine for bl.ocks.org. Give it a D3 function and you will see a list of relevant projects. Created by Irene Ros and Bocoup.

Data Visualization & D3.js Weekly Newsletter

  • A short curated weekly list of articles, reading, and videos on D3 and Datavis. Collected by Sebastian Gutierrez of Dashing D3js.

NYC D3.js group through Meetup.com

  • A monthly event focused on D3 and Datavis with speakers, socializing, and refreshments. Formerly organized by Sebastian Gutierrez, now run by Amit Schechter and Natalie Wesson of TWO-N.

Data Stories podcast

  • Possibly the best Datavis podcast, covering a wide breadth of guests and topics. Many episodes are interviews, some explore a specific topic, and increasingly more are breakdowns of a project alongside its authors. Hosted by Enrico Bertini and Moritz Stefaner.

Interactive Data Visualization for the Web

  • The definitive textbook for starting out in D3 (and JavaScript+HTML+CSS). Written by Scott Murray and sold through O'Reilly. Scott was cool enough to write a free interactive webapp version with live code! This is the book I used to get started. I knew basically no JavaScript before reading it, and he cut through the fog better than all the other textbooks I tried.

D3 for Beginners Part I: Joe Golike on Thinking in D3

  • An introduction to D3 from the perspective of a designer, by Joe Golike.

D3 for Beginners Part II: Vadim Ogievetsky on Developing in D3

  • A great next-step video for building your intuition for thinking in D3, by Vadim Ogievetsky.

Mark DiMarco: User Interface Algorithms [JSConf2014]

  • A deep-dive into JavaScript algorithms powering the UI and UX of charts and websites, by Mark DiMarco.
